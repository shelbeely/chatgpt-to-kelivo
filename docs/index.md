# ChatGPT to Kelivo

Welcome to the ChatGPT to Kelivo documentation! This project helps you export your ChatGPT conversations to [Kelivo](https://github.com/Chevey339/kelivo) and other applications.

## What is Kelivo?

[Kelivo](https://github.com/Chevey339/kelivo) is a powerful local AI assistant application that lets you manage and continue your AI conversations. This project provides tools to import your ChatGPT conversation history into Kelivo.

## Getting Started

1. **Export your ChatGPT data**: Go to [chatgpt.com](https://chatgpt.com) ‚Üí Settings ‚Üí Data Controls ‚Üí Export data
2. **Wait for email**: You'll receive an email with a download link
3. **Choose your method**: Use the standalone converter or browser extension
4. **Import to Kelivo**: Transfer the converted files to Kelivo
