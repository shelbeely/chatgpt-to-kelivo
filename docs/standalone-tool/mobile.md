# Mobile Usage Guide

The Standalone Converter is designed to work seamlessly on mobile devices. This guide provides tips and tricks for the best mobile experience.

## Why Use the Converter on Mobile?

- **No desktop required** - Convert your ChatGPT exports anywhere
- **Direct to [Kelivo](https://github.com/Chevey339/kelivo)** - Transfer files directly to Kelivo on your phone
- **Privacy** - All processing happens on your device

## Accessing the Converter on Mobile

### Option 1: Direct File Access

If you have the repository cloned or downloaded:

1. Navigate to the `standalone-tool` folder using your file manager
2. Tap on `index.html`
3. Choose to open it in your browser

### Option 2: Hosted Version

If a hosted version is available, simply visit the URL in your mobile browser.

## Tips for Mobile Users

### Downloading Your ChatGPT Export

1. Open [chatgpt.com](https://chatgpt.com) in your mobile browser
2. Request your data export as usual
3. When the email arrives, tap the download link
4. The ZIP file will download to your device

!!! tip "Finding Your Downloads"
    On iOS, check the Files app > Downloads folder.
    On Android, check your Downloads folder or notification bar.

### Uploading Files

The converter supports both tap-to-select and drag-and-drop:

- **Tap method**: Tap the upload area to open your file browser
- **Drag-and-drop**: On tablets, you can drag files from a file manager

### Handling Large Exports

On mobile devices with limited memory:

1. Keep only the converter tab open
2. Close other browser tabs to free memory
3. For very large exports, consider:
   - Processing on a tablet instead of a phone
   - Splitting the conversations.json file

### Downloading Results

When downloading converted files:

1. Single files download directly
2. ZIP downloads may prompt you to choose a save location
3. Files are typically saved to your Downloads folder

### Transferring to Kelivo

After downloading your converted .md files:

1. Locate the files in your Downloads folder
2. Open [Kelivo](https://github.com/Chevey339/kelivo)
3. Use Kelivo's import feature to add the conversations

## Browser-Specific Notes

### iOS Safari

- Works in iOS 14+
- Files download to the Files app
- Enable "Downloads" in Safari settings if prompted

### Chrome Mobile (Android)

- Full support for all features
- Files download to the Downloads folder
- Long-press files in Downloads to share directly

### Samsung Internet

- Full compatibility
- Uses the standard Downloads location

### Firefox Mobile

- Works on both iOS and Android
- May need to grant file access permissions

## Troubleshooting Mobile Issues

### File Won't Upload

- Check that you're selecting a .zip or .json file
- Try refreshing the page and re-selecting the file
- Ensure you have enough storage space

### Conversion is Slow

- Mobile devices have less processing power
- Large exports take longer
- Keep the browser tab active (don't switch apps)

### Download Fails

- Check your storage space
- Try downloading to a different location
- For large ZIP files, ensure stable connection

### Page Becomes Unresponsive

- Your export may be too large for mobile memory
- Try processing fewer conversations
- Consider using a desktop browser for very large exports

## Best Practices

1. **Stable Connection** - Use WiFi when downloading large exports
2. **Adequate Storage** - Ensure you have space for the export and converted files
3. **Battery** - Keep your device charged during large conversions
4. **Single Tab** - Close other browser tabs to maximize available memory
